OTPBox is an open-source, local-first TOTP two-factor (2FA) token manager.

Key features:
- Offline TOTP code generation, with standard otpauth import support
- Add accounts by camera scan or manual entry, organized and color-coded by category
- Locally encrypted storage (SQLCipher) with biometric unlock
- Auto-lock and sensitive clipboard (auto-cleared after a timeout)
- Backup and restore (export / import encrypted files, optional GitHub Gist sync)
- Fully usable offline, with no ads and no tracking
